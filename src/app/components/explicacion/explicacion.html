5. Explicación del Funcionamiento del Framework Angular

¿Sobre qué elementos actúa Angular?

Angular trabaja principalmente sobre una estructura basada en componentes, módulos y plantillas HTML. Estos elementos se combinan para crear interfaces dinámicas y reutilizables.

  • Componentes: Son las unidades básicas de construcción de la interfaz. Cada componente tiene su propia lógica, plantilla HTML y estilos.
  • Módulos: Agrupan componentes, servicios y otras funcionalidades. El módulo principal es App.
  • Servicios: Gestionan la lógica de negocio, las peticiones HTTP y funciones reutilizables.
  • Templates: Archivos HTML que definen la vista del componente.
  • Rutas: Permiten navegar entre páginas dentro de la aplicación.

¿Cómo se editan o modifican los ficheros en Angular?

Cada componente tiene una estructura compuesta por varios archivos:

La edición se realiza directamente en estos archivos:

  • .ts: Define la clase del componente y su comportamiento.
  • .html: Contiene la plantilla que se muestra en pantalla.
  • .css: Define estilos locales para ese componente.
  • .spec.ts: Define el archivo de pruebas unitarias.

Angular refresca los cambios automáticamente gracias al servidor de desarrollo iniciado con ng serve.

¿Cómo se relacionan los archivos dentro del framework?

Angular utiliza decoradores y etiquetas personalizadas para enlazar componentes entre sí. Cada componente tiene un nombre que lo identifica dentro de una plantilla HTML.

Por ejemplo, si un componente se declara con este nombre: 

nombre: 'hola-mundo'

Puedes insertarlo en otro HTML así: 

<hola-mundo></hola-mundo>

Esto permite construir interfaces modulares y organizadas.

¿Cómo se referencian archivos desde otras aplicaciones?

En angular en sí no se “importa” directamente desde otras aplicaciones, pero sí puede integrarse con:

  • APIs externas mediante servicios HTTP.
  • Microfrontends usando frameworks como Module Federation.
  • Bibliotecas compartidas vía npm.
  • Componentes externos importando módulos o librerías de terceros.
