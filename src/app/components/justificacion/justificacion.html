1. Justificación del Framework: Angular

¿Para qué sirve Angular?

Angular es un framework de desarrollo frontend creado por Google que nos permite construir aplicaciones web dinámicas, estructuradas y escalables. Destaca por su capacidad para construir aplicaciones de una sola página (SPA), ofreciendo una navegación fluida y una experiencia similar a las apps de escritorio.

¿Qué aplicaciones tiene?

Angular se utiliza principalmente para desarrollar aplicaciones web complejas, paneles administrativos, herramientas empresariales, sistemas de gestión, aplicaciones en tiempo real, plataformas PWA y, junto con Ionic, aplicaciones móviles. Su arquitectura modular lo hace ideal para proyectos de cualquier tamaño.

¿Sobre qué lenguajes se apoya?

Angular se basa en TypeScript como lenguaje principal.

TypeScript define un conjunto de tipos de JavaScript, lo que ayuda a los usuarios a escribir código JavaScript que es más fácil de entender. Todo el código TypeScript se compila con JavaScript y se puede ejecutar sin problemas en cualquier plataforma.

También utiliza HTML para las vistas y CSS o preprocesadores como SCSS para los estilos.

¿Qué prerrequisitos necesita?

Para trabajar con Angular es necesario tener conocimientos previos de JavaScript, HTML y CSS. Además, se debe instalar Node.js, ya que Angular CLI utiliza npm para gestionar dependencias y ejecutar comandos. También es recomendable familiarizarse con conceptos como componentes, módulos y servicios.
