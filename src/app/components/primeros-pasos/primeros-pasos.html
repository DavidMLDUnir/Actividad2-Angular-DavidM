3. Primeros Pasos con Angular

Crear una aplicación para mostrar “Hola Mundo”

Una de las primeras comprobaciones que podemos realizar al comenzar con Angular es crear un mensaje sencillo en pantalla, el clásico “Hola Mundo”.

Aunque Angular genera una aplicación inicial completa, es posible modificar el componente principal para mostrar nuestro propio mensaje.

Paso 1: Crear un nuevo proyecto

En la terminal, ejecuta:

ng new hola-mundo

Angular CLI generará toda la estructura base del proyecto. Luego entra dentro del directorio: 

cd hola-mundo

Si escribimos: 

code .

Se nos abrira nuestro edito de código Visual Studio Code y veremos nuestro proyecto como en la imagen:

Paso 2: Ejecutar la aplicación

Para iniciar el servidor de desarrollo, usaremos el comando:

ng serve

Por defecto, Angular se ejecutará en la URL: http://localhost:4200

Si todo está correcto, verás la aplicación inicial generada por Angular.

Paso 3: Modificar el componente principal

Para mostrar nuestro mensaje “Hola Mundo”, debemos editar el archivo: 

src/app/app.html

Reemplaza su contenido por:

<h1>Hola Mundo desde Angular</h1>

Guarda los cambios y la página del navegador se actualizará automáticamente mostrando el mensaje.
