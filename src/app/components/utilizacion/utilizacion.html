4. Utilización del Framework Angular

¿Cómo empezar a crear una aplicación base?

Como hemos visto en el apartado anterior angular genera automáticamente una estructura inicial muy completa al crear un nuevo proyecto. Esta aplicación base incluye componentes, módulos, archivos de configuración, un sistema de rutas opcional y un entorno preparado para compilar, probar y desplegar.

Tras ejecutar: 

ng new nombre-de-tu-aplicacion

Angular crea su estructura como hemos visto anteriormente.

Esta estructura permite comenzar a programar inmediatamente sin necesidad de configurar herramientas adicionales.

El componente principal (App)

El archivo app.html es la vista principal de la aplicación. Todo lo que se muestre por defecto aparece aquí.

Para personalizar la aplicación base, puedes editar este archivo y agregar tu contenido como hemos visto en los primeros pasos. 

<h1>Bienvenido a mi aplicación en Angular</h1>

Paso 4: Crear un componente personalizado “Hola Mundo”

También puedes crear un componente propio con este comando: 

ng generate component hola-mundo

Esto generará una carpeta con los archivos:

  • hola-mundo.ts
  • hola-mundo.html
  • hola-mundo.css
  • hola-mundo.spec.ts

Luego edita el archivo HTML generado hola-mundo.html:

<h2>¡¡¡¡Hola Mundo!!!!</h2>

Deberás, inserta el componente en app.html: 

<app-hola-mundo></app-hola-mundo>

¿Cómo visualizar los resultados de la ejecución?

Angular incluye un servidor de desarrollo que actualiza la página automáticamente cada vez que editas un archivo. Para ejecutarlo, usa: 

ng serve

Después, abre el navegador en:
http://localhost:4200

Desde ese momento, cualquier cambio en componentes, estilos o módulos se reflejará en tiempo real, facilitandote la creación de la aplicación.
